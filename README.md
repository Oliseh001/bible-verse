# Random Bible Verse and Quote API

## Overview
The **Random Bible Verse and Quote API** is a lightweight service built with **NestJS** that delivers random Bible verses and inspirational quotes on request. It’s designed for quick integration into applications such as websites, mobile apps, or other platforms.

## Features
- Fetch a **random Bible verse**.
- Fetch a **random inspirational quote**.
- Simple, fast, and stateless API.

---

## Endpoints

### 1. **`POST /bible-verse/verse`**
- **Description**: Returns a random Bible verse.
- **Request Body**: None required.
- **Response**: A plain text string containing a Bible verse.
  
#### Example:
**Request**:
```http
GET http://localhost:3000/bible-verse/verse
"For I know the plans I have for you, declares the Lord, plans for welfare and not for evil, to give you a future and a hope. - Jeremiah 29:11"

GET http://localhost:3000/bible-verse/quote
"The unexamined life is not worth living. - Socrates"

----------------------------------------------------
src/
├── app.module.ts              # Root module
├── bible-verse/
│   ├── bible-verse.controller.ts  # Handles API endpoints
│   ├── bible-verse.service.ts     # Logic for fetching Bible verses and quotes
